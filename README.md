# Java Unit Converter

## 📌 Description
This project is a GUI-based Unit Converter developed in Java using Swing. It allows users to convert values across different categories like Length, Mass, Numerical System, Temperature, and GST.

---

## 🚀 Features
- Simple and user-friendly interface
- Multiple conversion categories
- Dynamic unit selection
- Accurate conversion results
- Error handling for invalid input

---

## 🛠️ Technologies Used
- Java
- Swing (for GUI)
- AWT (for layout and event handling)

---

## ⚙️ How It Works
- The user selects a category (e.g., Length, Mass)
- Based on the selection, relevant units appear in dropdowns
- The user enters a value and clicks the **Convert** button
- The program processes the input and displays the result

---

## 💻 Code Overview
- Swing components are used to build the interface (buttons, text fields, dropdowns)
- Event handling is used to perform actions when the user interacts
- Conversion logic is implemented using a base unit method
- Input validation ensures only correct values are processed
